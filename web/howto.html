Basics

Keys are separated by a single space.
Everything written in a single line gets pressed and released at the same time.
To write text, use the TYPE function.

- It is used to put the cursor in the next line.
It is only used with TYPE
Example : TYPE Hello World!-
If TYPE contain any command and then - then it run automatically without ENTER key.

Example Explanation
GUI r Press the Windows key and the r key simultaneously
TYPE Hello World! Write Hello World!
